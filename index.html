Tips:

  1. First load a tSNE map
  2. You can select structures in the tSNE map or the ontology
  3. Hovering over the tSNE the ontology labels will be visible
  4. Right-mouse click on the tSNE map to select all related voxels
  5. In the ontology both tree or single selection is supported
